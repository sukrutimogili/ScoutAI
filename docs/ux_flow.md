# ScoutAI UX Flow

## 1. Landing Page
- Dashboard
- Upload Job Description
- Upload Resume Folder (Batch Upload)
- Hiring Run Name (Optional)
- Start Evaluation

## 2. Processing
- Show recruiter-friendly progress updates
  - Understanding the role
  - Analyzing resumes
  - Extracting relevant experience
  - Verifying supporting evidence
  - Preparing interview recommendations
  - Running fairness checks
  - Preparing recruiter summary
- Disable new hiring runs until processing completes
- Allow browser notifications (future)

## 3. Hiring Run Summary
- Total Candidates
- Recommendation Distribution
  - Strong Shortlist
  - Interview
  - Reject
  - Needs Review
- Fairness Alerts
- Candidates Requiring Validation
- Review Progress
- Actions
  - Review Candidates
  - Export Report

## 4. Candidate List
- Candidate Cards
- Sort
- Dynamic Filters
- Search (future)
- View Candidate

## 5. Candidate Review
- Recommendation
- Why?
- Needs Validation
- Interview Focus
- Supporting Evidence
- Similar Candidates

  - Compare
- Actions
  - Approve
  - Reject
  - Edit Recommendation
  - Request More Evidence

## 6. Candidate Comparison
- Recommendation
- Strengths
- Key Differentiators
- Needs Validation
